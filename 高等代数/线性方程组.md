## 一、背景回顾

### 1、什么是线性方程组？

线性方程组指形式如下的方程组

$\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = b_1\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = b_2\\...\\a_{s1}x_1+a_{s2}x2+...+a_{sn}x_n = b_s \end{cases}$

其中$x_1,...x_n$表示未知量，s是方程的个数，$a_{ij}$是方程的系数，$b_j$是常数项

* 方程组的解是$k_1,...k_n$带入x的位置使得每个等式成恒等式
* 方程组所有的解称为**解集合**
* 如果两个方程组有相同的解集合，称为**同解**的



### 2、求解线性方程组

#### 2.1、初中阶段求解

回顾初中阶段，线性方程组的求解方法——消元法

例如解方程组

$\begin{cases}2x_1-x_2+3x_3=1\\4x_1+2x_2+5x_3=4\\2x_1+x_2+2x_3=5\end{cases}$

> 第二个方程减去第一个方程的2倍，第三个减去第一个方程
>
> $\begin{cases}2x_1-x_2+3x_3=1\\4x_2-x_3=2\\2x_2-x_3=4\end{cases}$
>
> 第二个方程捡起第三个方程的2倍，互换第二，第三的顺序
>
> $\begin{cases}2x_1-x_2+3x_3=1\\2x_2-x_3=4\\x_3=6\end{cases}$
>
> 求得解为(9,-1,-6)



#### 2.1、初等变换

消元法不断对方程组进行变换，使用线性方程组的**初等变换**

* 非0数乘某一方程
* 把一个方程的倍数加到另一个方程
* 互换两个方程的位置

消元法即反复施行初等变换，消去其中的某些元素，最后形成一个梯形方程组



#### 2.3、消元法求解

消元法处理线性方程组，最后形成一个梯形方程组

$\begin{cases}c_{11}x_1 +c_{12}x_2+...+c_{1n}x_n = d_1\\c_{22}x_2+...+c_{2n}x_n = d_2\\...\\c_{rr}x_r+...+c_{rn}x_n=d_r\\0=d_{r+1}\\0=0...\\0=0\end{cases}$

如果$d_{r+1}\neq 0$则方程组无解。

如果r=n，有唯一解

如果r<n，不是唯一解



#### 2.4、r<n求解

$\begin{cases}c_{11}x_1 +c_{12}x_2+...+c_{1n}x_n = d_1\\c_{22}x_2+...+c_{2n}x_n = d_2\\...\\c_{rr}x_r+...+c_{rn}x_n = d_r\end{cases}$

改写成

$\begin{cases}c_{11}x_1 +c_{12}x_2+...+c_{1r}x_r = d_1-c_{1,r+1}x_{r+1}-...-c_{1n}x_n\\c_{22}x_2 +c_{23}x_3+...+c_{2r}x_r = d_1-c_{2,r+1}x_{r+1}-...-c_{2n}x_n\\...\\c_{rr}x_r = d_r-c_{r,r+1}x_{r+1}-...-c_{rn}x_n\end{cases}$

任给一组$x_{r+1},...x_n$，就可以唯一确定$x_1,x_2,...x_r$的值。

把$x_1,x_2,...x_r$通过$x_{r+1},...x_n$表示出来的表达式称为方程组的**一般解**

把$x_{r+1},...x_n$称为**自由未知量**



### 3、齐次线性方程组

齐次线性方程组指形式如下的方程组

$\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = 0\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = 0\\...\\a_{s1}x_1+a_{s2}x2+...+a_{sn}x_n = 0 \end{cases}$

如果s<n，即方程组的数量小于未知数的个数，则方程组一定有非零解。



## 二、矩阵的使用

### 1、行列式与矩阵

#### 1.1、行列式求解方程组

在前一章行列式的学习中，知道克拉默法则求解线性方程组，不过要求方程组未知数个数与方程个数一样

$\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = b_1\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = b_2\\...\\a_{n1}x_1+a_{n2}x2+...+a_{nn}x_n = b_n \end{cases}$

如果行列式d不等于0，方程组有唯一解`可以作除法了`

方程的解$x_i = \frac{di}{d}$，如果未知数个数多的话计算量还是比较大的。



不过不可能永远都是方程组未知数个数与方程个数一样，需要推出新的方法。所以后来推出了矩阵来求解。

#### 1.2、为了省力的衍生

行列式作为一种工具，是伴随线性方程组的求解而产生的。

回顾我们一开始的行列式产生，我们发现在求解线性方程组的时候那些$x_i$一点用都没有，但每次变换都要抄上运算。

数学家们表示抄的烦死了，发明了行列式，但行列式只能面向方程组个数等于变量个数的情况，对于其他情况不满足。

因此提出了矩阵，解决方程组未知数个数与方程个数不一样的情况。

#### 1.3、增广矩阵

线性方程组

$\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = b_1\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = b_2\\...\\a_{s1}x_1+a_{s2}x2+...+a_{sn}x_n = b_s \end{cases}$

对应的**增广矩阵**为

$\begin{bmatrix}a_{11}&a_{12}&...&a_{1n}&b_1\\a_{21}&a_{22}&...&a_{2n}&b_2\\...&..&...&...&...\\a_{s1}&a_{s2}&...&a_{sn}&b_s\end{bmatrix}$

对增广矩阵做**初等变换**就不用抄一堆$x_i$了。



#### 1.3、系数矩阵

对于齐次线性方程组，b都是0也可以不用抄的，能懒就懒，提出了系数矩阵。

$\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = 0\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = 0\\...\\a_{s1}x_1+a_{s2}x2+...+a_{sn}x_n = 0 \end{cases}$

对应的系数矩阵为：

$A = \begin{bmatrix}a_{11}&a_{12}&...&a_{1n}\\a_{21}&a_{22}&...&a_{2n}\\...&..&...&...\\a_{s1}&a_{s2}&...&a_{sn}\end{bmatrix}$

但这样还是要写好几个数字，数学家们表示还是懒得写，提出了向量，用**a**表示，那么$A = [a_1,a_2,...,a_n]$，其中$a_i = (a_{1i},a_{2i},...,a_{si})$



### 3、线性方程组

#### 3.1、矩阵的消元法

为了求解线性方程组提出了一个新概念——矩阵，那么原本线性方程组的操作也是要在矩阵上实现的。

> 有线性方程组
>
> $\begin{cases}a_{11}x_1+a_{12}x2+...+a_{1n}x_n = b_1\\a_{21}x_1+a_{22}x2+...+a_{2n}x_n = b_2\\...\\a_{s1}x_1+a_{s2}x2+...+a_{sn}x_n = b_s \end{cases}$
>
> 消元法处理线性方程组，最后形成一个梯形方程组
>
> $\begin{cases}c_{11}x_1 +c_{12}x_2+...+c_{1n}x_n = d_1\\c_{22}x_2+...+c_{2n}x_n = d_2\\...\\c_{rr}x_r+...+c_{rn}x_n=d_r\\0=d_{r+1}\\0=0...\\0=0\end{cases}$
>
> 如果$d_{r+1}\neq 0$则方程组无解。
>
> 如果r=n，有唯一解
>
> 如果r<n，不是唯一解

对应到矩阵上，本质就是$x,y$懒的抄了，除了$xy$以外的要计算的数值全部抄下来形成矩阵，然后对矩阵做**初等变换**

如果r=n，$c_{nn}=d_n$，继而推出其他值，有唯一解（行列式有值，克拉默法的分子也有值）

$A1 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1n}&d_1\\0&c_{21}&...&c_{2n}&d_2\\...&..&...&...&...\\0&0&...&c_{nn}&d_n\end{bmatrix}$

如果r<n，不是唯一解

$A2 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1r}&...&c_{1n}&d_1\\0&c_{21}&...&c_{2r}&...&c_{2n}&d_2\\...&..&...&...&...&...&...\\0&0&...&c_{rr}&...&c_{rn}&d_r\end{bmatrix}$



#### 3.2、方程组有解：r(A) = r($\hat A$)

当$d_{r+1}$不等于0，方程组无解，即系数矩阵的秩小于增广矩阵的秩，r(A) < r($\hat A$)

当r=n，方程组有解，即系数矩阵的秩n等于增广矩阵的秩n，r(A) = r($\hat A$) = n

当r<n，方程组有解，即系数矩阵的秩r等于增广矩阵的秩r，r(A) = r($\hat A$) = r

即方程组有解的条件为系数矩阵和增广矩阵有相同的秩



#### 3.2、齐次线性方程组

增广矩阵初等变化后：

如果r=n，推出$c_{nn}=0$，继而推出全是零解

$A1 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1n}&0\\0&c_{21}&...&c_{2n}&0\\...&..&...&...&...\\0&0&...&c_{nn}&0\end{bmatrix}$

如果r<n，方程组一定有非零解

$A2 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1r}&...&c_{1n}&0\\0&c_{21}&...&c_{2r}&...&c_{2n}&0\\...&..&...&...&...&...&...\\0&0&...&c_{rr}&...&c_{rn}&0\end{bmatrix}$

不过齐次线性方程组我们一般使用系数矩阵的，毕竟能不抄0就不抄0了

$A3 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1n}\\0&c_{21}&...&c_{2n}\\...&..&...&...\\0&0&...&c_{nn}\end{bmatrix}$

$A4 = \begin{bmatrix}c_{11}&c_{12}&...&c_{1r}&...&c_{1n}\\0&c_{21}&...&c_{2r}&...&c_{2n}\\...&..&...&...&...&...\\0&0&...&c_{rr}&...&c_{rn}\end{bmatrix}$



#### 3.3、齐次方程组有解的条件：行秩r<n

矩阵A3是经过齐次方程组经过初等变换后的梯形矩阵，秩为n，方程组只有零解；

矩阵A4是经过齐次方程组经过**初等变换**后的梯形矩阵，秩为r，方程组有解；

所以齐次方程组的系数矩阵的**行秩**r<n，则方程组有解。

> 为什么这里是行秩，不是列秩？
>
> 回顾增广矩阵的初等变换，都是行之间的操作，到现在为止没有出现列之间的操作
>
> * 非0数乘某一方程
> * 把一个方程的倍数加到另一个方程
> * 互换两个方程的位置



#### 3.4、齐次线性方程组的有解条件扩展

（1）矩阵的秩r<n

矩阵有性质：矩阵的行秩与列秩是相等的

所以求解系数矩阵的列秩r，如果r<n，那么对应的齐次方程组有解

`到这里就不再局限于行秩了`



（2）系数矩阵行列式为0，齐次线性方程组有解

当齐次线性方程组的系数矩阵是个$n*n$的方阵A。

如果A的秩为n，即A经过初等变换变成一个方方正正的上三角矩阵，此时的行列式不等于0

如果A的秩为r<n，即A经过初等变换变成一个梯形矩阵，好几行都是0，构成方阵，此时的行列式为0

## 三、解的结构

### 1、齐次线性方程组

#### 1.1、解的线性组合

齐次线性方程组具有特点：解的线性组合还是方程组的解

（1）两个解的和还是方程组的解

> $(k_1,k_2,...,k_n)$与$(l_!,l_2,...,l_n)$是两个解，即带入方程为0
>
> 和带入方程后也可以算出结果为0，即和为方程组的解

（2）一个解的倍数还是方程组的解

所以，如果齐次线性方程组有解，这些解的线性组合还是方程的解



#### 1.2、基础解系

齐次线性方程组的一组解$\eta_1,\eta_2,...,\eta_t$称为一个基础解系，如果

（1）齐次线性方程组的任意解都可以表示成$\eta_1,\eta_2,...,\eta_t$的线性组合

（2）$\eta_1,\eta_2,...,\eta_t$线性无关`谁也别想干掉谁`



#### 1.3、齐次线性方程组的解

在齐次线性方程组有非零解的情况下，有基础解析，并且基础解析所含的个数为n-r个

### 1、类梯形上三角

首先把矩阵经过一系列初等变换变成一个类梯形的上三角矩阵。

（1）保障对角线(i,i)上的值不为0

（2）如果为0则和其他行做交换，看看有没有绝对值得1的行，没有就和下一行交换

（3）把一个方程的倍数加到另一个方程，倍数=-(j,i)/(i,i)

（4）最后就变成一个类梯形的矩阵了
